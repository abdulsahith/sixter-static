502

Bad Gateway

Your local server is not responding.

What happened?

The web server reported a bad gateway error. Your local server is not responding or is unreachable through the Sixter tunnel. The Sixter network successfully connected, but couldn't reach your application.

What can I do?

If you are a visitor:

Please try again in a few minutes. If the problem persists, contact the owner of this tunnel.

If you are the owner:

  • Ensure your local application is running on the specified port
  • Verify the Sixter tunnel is active with: sixter --subdomain <subdomain> --port <port>
  • Check that your firewall isn't blocking the connection
  • Restart your local server and tunnel connection
  • Make sure the port number matches your running application
